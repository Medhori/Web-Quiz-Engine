# Web Quiz Engine
Web Quiz Engine is a multi-users web service for creating and solving quizzes. Created with spring framework.

## Prerequisites
This program requires at least Java 11 to compile and run properly.

## Dependencies
- Spring Web
- Spring Security
- Spring Data JPA
- Spring Boot Actuator
- H2 Database
